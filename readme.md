## ðŸš€ Getting Started

1. **Clone this repo:**
    ```sh
    git clone https://github.com/Nirmitkhurana10/portfolio.git
    cd portfolio
    ```

2. **Install dependencies:**
    ```sh
    npm install
    ```

3. **Run locally:**
    ```sh
    npm run dev
    ```

4. **Open in your browser:**
    ```
    http://localhost:3000
    ```
